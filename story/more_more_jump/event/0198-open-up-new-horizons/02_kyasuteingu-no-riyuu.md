# 2. キャスティングの理由

みのり: ええ～！？ 愛莉ちゃんが、『謎カフェ』のドラマに……！？
愛莉: ええ。いろいろ考えたんだけど…… オファーを受けたいなって思ってるの
遥: 『謎解きカフェタイム』って、 たしか人気漫画を実写化した連続ドラマだよね
斎藤: はい。原作漫画の累計発行部数は、1500万部超え——
斎藤: ドラマのほうは、シーズン３の放送と同時に 劇場版の制作も発表されています
雫: そういえば……この前現場で会ったスタイリストさんも、 その『謎カフェ』の話をしていたわ
雫: 配信サイトで見始めたんだけど、 面白くてつい夜更かししちゃうって
愛莉: わかるわ。わたしも依頼をもらってから 勉強のために見始めたけど、 ついつい夢中になっちゃったもの
みのり: うちのクラスにも、 ドラマにすっごくハマってる子がいるよ！
みのり: 休み時間に、おすすめのシーンを 見せてもらったんだけど…… たしか、移動式のカフェのお話だよね？
愛莉: そうよ。主人公は、そのカフェの店主なんだけど…… ちょっと変わった人で、謎解きが大好きなの
雫: あら。なんだか探偵さんみたいね
愛莉: ふふ、たしかにね。 でも店主は、そこまで大きな謎を求めてるわけじゃなくて…… 日常の中の、小さな謎を解いていくの
遥: 小さな謎？
愛莉: ええ。たとえば、子供がなくしちゃった おもちゃのある場所を当てたりとか——
愛莉: 昔誰かに言われて引っかかった言葉が、 本当は全然違う意味だったことに気づかせたりとか…… そういうささいなことね
愛莉: でも、その謎が解けることで、 お客さんはちょっとだけ気持ちが軽くなったり、 前に進めるようになるの
雫: ……なんだか素敵ね
愛莉: でしょ？
愛莉: ただ——店主にも何か秘密があるみたいで、 ずっと同じ場所にはいないって決めてるのよね
愛莉: だから、１話が終わる度に別の街へ行って、 また新しいお客さんに出会うのよ
遥: なるほど。毎回、どんどん新しいゲストが 出てくるタイプの話なんだね
みのり: じゃあ、愛莉ちゃんはお客さんの役なの！？
愛莉: 違うわ。わたしは、その友達の役なの
愛莉: ゲスト側のメインではないけど…… セリフとか登場シーンも、かなり多いほうではあるわね
みのり: おお～！
遥: すごいな……。 過去の出演者がまとめてあるサイトを見ると、 ゲストまで含めて演技派の俳優さんばっかりだね
愛莉: そうなのよね……
愛莉: だからこそ、わたしも受けるかはちょっと悩んだのよ
みのり: どうして？
愛莉: そりゃあ、この作品のファンにとって ネガティブな印象になるかもしれないからよ！
愛莉: こんな豪華なメンツの中に、 多少ドラマに出たことはあるとはいえ…… 演技経験がほとんどないわたしが入るのよ？
愛莉: ファンとしては、ちょっと嫌じゃないかしら？
雫: あ……
遥: たしかに、演技経験がないアイドルが いきなり担当することになったら、心配になる人はいるかもね
遥: でも……ドラマの制作側も、そういうリスクがあるのは 理解してるはずじゃないかな
みのり: それって……？
遥: つまり——そういうリスクがあったとしても、 愛莉に声をかける理由があるってことだと思う
斎藤: ……私も気になって調べてみたんですが、 『謎カフェ』は人気がある一方で、コアの視聴者層が だんだんと固定されてきているみたいなんです
斎藤: なので、新しい視聴者を獲得するために 話題性のあるゲストを積極的に 起用しようとしているんじゃないかと
愛莉: この業界だと、よく聞く話ね
愛莉: ただ、そういう理由があるにしても わたしに声がかかるには弱い気がするのよね。 だから、思い切って先方に聞いてみたの
愛莉: わたしに依頼しようと思ってくれた、決め手はなんなのか。 そうしたら……
みのり: そうしたら……？
愛莉: ……漫画を見ながら説明するのが早いわね
愛莉: ほら、この子がわたしにオファーが来た子よ
みのり: わ……！　かわいい！
雫: ふふ、本当ね。 ……それに、この子……
雫: ……やっぱり。愛莉ちゃんに似てる気がするわ
みのり: あ……たしかに……！？
愛莉: 実際、それが決め手になったみたいなの。 メイクとか、演じ方次第でそっくりになるはずだって 力説されたわ
遥: なるほどね。 原作のイメージに合わせてのキャスティングなんだ
愛莉: ええ。——ドラマの制作側は、 過去にわたしがやった演技の仕事も見たうえで オファーをくれたそうなの
愛莉: たしかに演技の経験は浅いかもしれないけど、 演じられるポテンシャルはありそうだって そう期待してくれてるみたい
斎藤: ——愛莉ちゃん、依頼の話を聞いてすぐに 原作の漫画を全巻買いに行ったんですよね
みのり: えっ！？
愛莉: ドラマも、シーズン１から履修中よ。 引き受けるかどうか決めるのに、 相手のことを知らないで判断できないもの
愛莉: そしたら——どの話もすっごく面白くて、 感動したのよね！
雫: 愛莉ちゃん……
愛莉: ファンの声にも目を通したけど、 みんな、すごく熱量が高かったの
愛莉: この作品に元気をもらったとか、 自分も頑張ってみる勇気をもらえたとか、 そういうコメントがたくさんあって……
愛莉: ステージとはまた違った形で、 たくさんの人に希望を届けられる仕事だと思ったの
愛莉: もちろん、プロの俳優さんにまじって 演技をするのは、かなり大変だと思うけど——
愛莉: それでも、挑戦してみたいと思ってるわ
遥: ……うん。いいと思うな
みのり: 愛莉ちゃんなら、絶対にできるよ！
斎藤: 私も、全力でサポートします！
愛莉: ありがとう、みんな
みのり: そういえば、ドラマで愛莉ちゃんがやる役って どんな感じの子なの？
愛莉: どんな感じ？
みのり: うん！　性格とか、設定とか！
愛莉: 原作を読んでもらうのが一番早いでしょうけど、 性格の話なら——
愛莉: わたしとは、正反対って感じの子かしら？
